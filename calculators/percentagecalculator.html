Common Percentage Formulas

Percentage of a Number = (Percentage ÷ 100) × Number

Percentage Increase = ((New Value - Old Value) ÷ Old Value) × 100

Percentage Decrease = ((Starting Value - Ending Value) ÷ Starting Value) × 100

What Percent = (Part ÷ Whole) × 100

Frequently Asked Questions (FAQ)

To find X% of Y, divide the percentage by 100 and multiply it by the number.

For example, 20% of 50 is:
(20 ÷ 100) × 50 = 10
To calculate percentage increase, subtract the old value from the new value, divide by the old value, and multiply by 100.

Example:
Old value = 100, New value = 125
Increase = ((125 - 100) ÷ 100) × 100 = 25%
To calculate percentage decrease, subtract the ending value from the starting value, divide by the starting value, and multiply by 100.

Example:
Starting value = 80, Ending value = 60
Decrease = ((80 - 60) ÷ 80) × 100 = 25%
