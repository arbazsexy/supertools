How to Use the Open Graph Generator

The Open Graph Generator helps you create the Open Graph meta tags used by social platforms to generate rich link previews. These tags control how your page appears when someone shares it on platforms like Facebook, LinkedIn, WhatsApp, and other services that support social preview metadata.

To use the tool, enter your page title, description, URL, and image URL. Then select the content type and optionally add your site name, locale, and image dimensions. Once the fields are filled, click Generate OG Tags to create copy-ready HTML code.

The tool also shows a simple preview of how your page may look in a shared social card. This helps you spot missing information and improve the appearance of your shared links before publishing.

What Are Open Graph Tags?

Open Graph tags are metadata tags placed inside the <head> section of a webpage. They help social platforms understand your content and build a better link preview using values such as title, description, image, URL, and type.

Add the generated Open Graph tags inside the <head> section of your HTML page.
