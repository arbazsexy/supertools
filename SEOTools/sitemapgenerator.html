How to Use the Sitemap Generator

The Sitemap Generator helps you create a valid XML sitemap for your website. A sitemap tells search engines which URLs exist on your site and can help them crawl and discover your pages more efficiently.

To use this tool, enter your page URLs or paths one per line. You can also add a base URL so relative paths like /about or /contact automatically become full URLs. Then choose a change frequency, set a priority, and optionally include a last modified date.

Once everything is ready, click Generate Sitemap. The tool will create the full XML sitemap code for you, remove duplicate URLs if selected, and optionally sort the URLs alphabetically. You can then copy the generated code and save it as sitemap.xml.

What is an XML Sitemap?

An XML sitemap is a structured file that lists important URLs on your website. It helps search engines understand your site structure and may improve how efficiently your pages are crawled and indexed.

Common sitemap location: https://yourdomain.com/sitemap.xml

Who Can Use This Tool?
