How To Use This PDF Tools Hub

This page works best as a starting point. Open the PDF tool that matches your task first, then follow the related links inside that tool page if your document needs another step.

  • Use Merge PDF and Split PDF when page order, document grouping, or page extraction is the main task.
  • Use Rotate PDF before merging scanned documents so every page faces the right direction.
  • Use Compress PDF after editing when file size matters for email, forms, portals, or uploads.
  • Use Unlock PDF only for files you own or have permission to edit, then continue with splitting, rotating, merging, or watermarking.
  • Use Add Watermark to PDF when a document needs DRAFT, CONFIDENTIAL, SAMPLE, INTERNAL, or COPYRIGHT labels.

Do PDF tools upload my files to a server?

★FAQ

Which PDF tool should I start with?

Start with Merge PDF to combine files, Split PDF to extract pages, Compress PDF to reduce size, Rotate PDF to fix orientation, Unlock PDF for authorized protected files, and Add Watermark to PDF for document labels.

🔁FAQ

Can I use multiple PDF tools together?

Yes. A common workflow is to rotate scanned pages, merge corrected files, compress the final PDF, and add a watermark before sharing.

🔑FAQ

Can the Unlock PDF tool handle every encrypted file?

No browser workflow can guarantee every encrypted PDF. Use Unlock PDF only for files you own or have permission to edit, and only when the file can be opened successfully.
